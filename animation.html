  • animation
  • animation-name: 애니메이션의 이름
  • animation-duration: 진행 시간
  • animation-delay: 지연 시간
  • animation-iteration-count: 반복 횟수(무한반복은 infinite)
  • animation-direction: 진행 방향
    1. normal
    2. reverse
    3. alternate
    4. alternate-reverse
  • animation-timing-function 가속 감속 등의 효과
    1. ease
    2. ease-in
    3. ease-out
    4. ease-in-out
    5. linear
    6. cubic-bezier(n,n,n,n) 베지에곡선값 생성기
  • animation-fill-mode: 애니메이션 정지시의 상태 지정
    1. none
    2. forwards:마지막 키프레임의 상태
    3. backwards:첫 키프레임의 상태
    4. both:둘다
  • animation-play-state: running/paused; 애니메이션을 진행/멈춤 상태로 지정.
  1. none
  2. forwards
  3. backwards
  4. both

animation-direction: 진행 방향

  1. normal: 기본값. 정방향으로 진행(0%~100%)
  2. reverse: 역방향으로 진행(100%~0%)
  3. alternate: 왕복 진행(0%~100%~0%)
  4. alternate-reverse: 역방향 왕복 진행(100%~0%~100%)
