margin

margin?
테두리(border)를 기준으로 바깥쪽에 생기는 4방향의 여백.
한줄에 나란히 오는 요소(inline요소)들은 상하방향의 margin이 적용 안 됨.
  1. margin:n n n n; 바깥여백이 top right bottom left의 순서대로 각각 적용
  2. margin:n n; 바깥여백이 top와 bottom, right와 left가 한쌍으로 같이 적용
  3. margin:n; 바깥여백이 양사방 동일하게 적용
    • margin-top:n; 위쪽 바깥여백이 적용
    • margin-right:n; 우측 바깥여백이 적용
    • margin-bottom:n; 아래쪽 바깥여백이 적용
    • margin-left:n; 좌측 바깥여백이 적용

1. margin:n n n n;

margin:50px 0 0 0;
margin: 10px 50px 100px 150px;
margin: 0 100px 100px 50px;
margin: 100px 30px 100px 50px; margin: 200px 50px 300px 10px; margin: 500px 100px 500px 120px;

2. margin:n n ;

margin: 100px 10px;
margin: 10px 100px;
margin:100px 10px; margin: 10px 100px;

3. margin:n ;

margin: 50px;
margin: 30px;
margin:50px; margin:30px ;
