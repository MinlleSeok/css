text- 로 시작하는 글자속성

  • text-align : left/center/right/justify ; 컨텐츠의 수평방향을 지정
  • text-decoration : none/underline/overline/line-through; 텍스트의 줄
  • text-indent : npx/nem/n%/... ; 들여쓰기(음수값 가능)
  • text-transform : uppercase,lowercase,capitalize ; 영문의 대소문자 지정
  • text-shadow :npx npx npx 색상; 텍스트의 그림자 지정
  • text-overflow : ;
