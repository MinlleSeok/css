transform:변형함수();

  • translate 위치 변형
    1. transform:translateX(n); X축 방향. n은 모든 단위 가능.(음수값 가능)
    2. transform:translateY(n); Y축 방향. n은 모든 단위 가능.(음수값 가능)
    3. transform:translate(n,n); X축,Y축방향. n은 상동
  • scale 크기 변형
    1. transform:scaleX(n); X축 방향. n은 1을 기준으로 1보다 작으면 축소, 1보다 크면 확대
    2. transform:scaleY(n); Y축 방향. n은 상동
    3. transform:scale(n,n); X축,Y축방향. n은 상동
  • rotate 회전 변형
    1. transform:rotateX(n); X축 방향. n의 단위는 deg.(음수값 가능)
    2. transform:rotateY(n); Y축 방향. n은 상동
    3. transform:rotate(n); Z축 방향. n은 상동
  • skew 기울기 변형
    1. transform:skewX(n); X축 방향. n의 단위는 deg.(음수값 가능)
    2. transform:skewY(n); Y축 방향. n은 상동
    3. transform:skew(n,n); X축,Y축 방향. n은 상동

transform의 다중 적용

변형을 2가지 이상 적용하려면 각 변형 함수를 나열하면 된다.
(예시) transform:변형함수1() 변형함수2() 변형함수3();
단, 순서에 따라 결과가 달라질 수 있음을 유념.

transform: translate rotate; 일 때

translateX(800px) rotate(720deg)

transform: rotate translate; 일 때

rotate(720deg) translateX(800px)

transform: translate scale ; 일 때

translate(700px,-300px) scale(4)

transform: scale translate ; 일 때

scale(4) translate(700px,-300px)

transform-origin : 변형의 기준점

변형의 기준점은 center center 또는 50% 50% 가 기본값임.
수평과 수직의 순서로 변경 가능함.
(예시1) transform-origin : n% n% / npx npx / ...
(예시2) transform-origin : left top / right bottom/ ...

scale변형

transform-origin:center center;
transform-origin:center top
transform-origin:center bottom
transform-origin:10% 30%;

rotate변형

transform-origin:center center;
transform-origin:center top
transform-origin:center bottom
transform-origin:10% 30%;
transform-origin:0% 0%;
