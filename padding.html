padding

padding?
테두리(border)를 기준으로 안쪽에 생기는 4방향의 여백
  1. padding:n n n n ;
    상 우 하 좌의 순서대로 각각 값이 다르게 적용됨.
  2. padding:n n ;
    상하 좌우가 한쌍으로 값이 서로 같게 적용됨.
  3. padding:n ;
    양사방의 값이 모두 같게 적용됨.
    • padding-top:n;
    • padding-right:n;
    • padding-bottom:n;
    • padding-left:n;
